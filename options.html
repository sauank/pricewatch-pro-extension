📊 Google Sheets Sync
Logs every price drop to a Google Sheet. Requires Google OAuth setup in manifest.json. Setup steps ↓
Find it in the sheet URL: docs.google.com/spreadsheets/d/[THIS PART]/edit
Google OAuth Setup (one-time):
  1. Go to Google Cloud Console
  2. Create a new project → Enable Google Sheets API
  3. Go to Credentials → Create OAuth 2.0 Client ID → select Chrome Extension
  4. Set Application ID to your extension's ID (find at chrome://extensions)
  5. Copy the Client ID and paste it in manifest.json under "oauth2" → "client_id"
  6. Reload the extension, then click Authorize above
